# sample-ci

The simplest CI example for Node.js

## Installation

1. Install the Node package manager:

    ```bash
    brew install npm
    ```

2. Install Jest (a popular JavaScript Testing Framework):

    ```bash
    npm install --save-dev jest
    ```

3. Check the install by running the tests:

    ```bash
    npm test
    ```
